## 사진 피드 앱

## 실행 명령어

`npm start`

## 디렉타리 구조

```bash
├── README.md
├── craco.config.js # craco 설정
├── jsconfig.json # 절대경로 alias 설정
├── package-lock.json
├── package.json
├── public #public 파일
│   ├── icons
│   ├── index.html
└── src
├── api #API관련
├── assets #자원
├── constants #상수
├── index.js
├── utils #유틸
└── views #뷰 컴포넌트
```
